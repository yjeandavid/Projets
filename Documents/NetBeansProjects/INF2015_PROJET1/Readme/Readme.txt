INF2015_PROJET1Fonctionnement:	Le programme necessite 2 fichiers au format JSON pour fonctionner.	Le premier fichier represente le fichier source (fichier contenant les donnees),	le second fichier contiendra les resultats du traitement du fichier source s'il	est traitable.	Test	Pour tester le programme depuis NetBeans, veuillez modifier les proprietes du 	projet dans l'onglet Run-> arguments.	Un fichier JSON accompagne le programme.	Dependances:	Les differentes librairies suivantes:		- commons-lang-2.5.jar		- commons-logging-1.1.3.jar		- commons-collections-3.2.1.jar		- commons-beanutils-1.9.1.jar		- ezmorph-1.0.6.jar	sont necessaires pour le bon fonctionnement de la librairie json-lib-2.4-jdk15.jar	La librairie commons-io-2.4.jar permet de lire un fichier et convertir tout le	contenu en String
